Visit 5 - D6011 - Second Stage Implant Surgery (use this code for each implant placement)

Visit 6 - D6065 - Implant Supported ACC (use this code for ACC implant crown)

Visit 8 - D2701.T - Implant Crown Seat (use this code for each crown seat)
